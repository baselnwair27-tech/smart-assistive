Blind Status

This screen shows the caregiver menu with the blind status option. In the app, pressing this opens status details and alerts for the user who needs assistance.

Emergency History

This screen displays previous alerts and timestamps. It helps caregivers track events and review past emergency responses.

Device Status

This screen reports whether the device is active and connected. It shows current device health and WiFi status so caregivers know the system is working.

AI Caregiver Chat

This screen is the app’s chat assistant. Caregivers can ask about emergency alerts, location monitoring, safe zones, and how to use the system.

Contact Us

This screen provides contact options: email, phone, Instagram, and Facebook. It makes it easy to reach support whenever needed.

Settings

This screen lets users select the app theme and personalize the experience. In the mobile app, it appears as the final option in the caregiver menu.
